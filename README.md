# Guide D'utilisation
Vous avez 2 méthodes. La première est d'avoir le programme sur votre OPC (recommandé). La deuxième est de l'utiliser via le Web.

## Utilisation via web
Rendez vous sur un IDE python en ligne comme https://www.online-python.com/ et collez le code suivant:
``ac = int(input("Entrez le nombre d'ACs: "))
cd = int(input("Entrez le nombre de CDs: "))

if ac >= (cd / 3):
    print("L'article 5 s'applique.")
else:
    print("L'article 5 ne s'applique pas.")``
Vous pouvez ensuite executer le code.

## Utilisation via PC
1. Installer python via https://python.org/
2. Une fois python installé, télécharger le code source sur la page sur laquelle vous lisez ceci.
